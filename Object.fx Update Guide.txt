
Ver 1.1 -> Ver 1.11

	Object.fx from ver 1.1 can be used without any modification, but if you want to use the new feature, add the new line
	
		/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
		...
		
		#define SpecularLight_Intensity 1
		#define SpecularLight_Focus 50
		#define SpecularLight_Affected_By_LightDirection 1 //1 is enabled (MMD default), 0 is disabled <-- NEW LINE
		
		/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
		
	(If the fx doesn't have this line, then the shader will automatically set it as 1 by default)
